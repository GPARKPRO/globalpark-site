# Tokenomics

GPARK utilizes a native utility token to support ecosystem activities.

## Token Distribution

- 40% Community Incentives
- 25% Ecosystem Development
- 20% Founders and Team
- 10% Reserve
- 5% Advisors

## Utility

- Access to premium features
- Governance voting rights
- Staking rewards

Transparent and sustainable economics fuel the project's growth.
